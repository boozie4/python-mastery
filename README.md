# Python Mastery — 30-Day Training Program

A structured, hands-on program to take you from Python basics to expert-level proficiency.

## How This Works

Each day has a folder inside the relevant week. Inside each day folder:

| File | Purpose |
|------|---------|
| `lesson.md` | Read this first — concepts explained clearly |
| `exercises.py` | Your workspace — fill in the `TODO` sections |
| `solutions.py` | Reference answers — only peek after you attempt |

**Workflow each day:**
1. Read `lesson.md`
2. Work through `exercises.py` in Claude Code or your terminal
3. Run your code: `python exercises.py`
4. When done, update `PROGRESS.md` and commit to GitHub
